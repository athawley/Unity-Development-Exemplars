This example creates a cone of vision that highlights what the player (and enemy) object can see.

This example uses the the scene and scripts below as a starting point to add the field of view onto.

Examples\PlayerMovement\PlayerMovementTwinStick

Note that the Custom FOV added to the Editor is in the Editor folder of assets.
All changes to the Unity Editor UI need to be in the Editor folder

Editor\FieldOfViewEditor
